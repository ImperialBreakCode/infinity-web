About

The website is divided into two parts - the main site here and Instagram, which is just an imitation I made for fun and I'm not making any money of it so Mark we gucci? In the main site you can create articles and in Instagram you can create posts and comment on them. You also have settings like switching to dark theme, changing following and followers count and etc. Of course you need an account to do all that but the register page is not public sooo.

Front End

  • HTML + jinja2 templates
  • CSS + Bootstrap 5
  • Javascript + JQuery and JQuery UI
  • Javascript libraries:
    • image cropping - Cropper.js
    • 3D rendering - Three.js for
    • emoji picker - Emoji-button.js
  • images from unsplash or created with photoshop
  • vector images from unDraw and other places

Back End

  • Language - Python
  • Framework - Flask
  • Some libraries:
    • Ckeditor
    • SQLAlchemy
    • WTForms

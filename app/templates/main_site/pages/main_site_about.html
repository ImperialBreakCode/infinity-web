About

The website is divided into three parts - the main site (which you are currently on), a redesigned Instagram with a few functions, and an experimental website called Solaris. The main site allows you to create articles (ehh..) and in Instagram you can create posts, comment on them, change the UI to a dark theme and etc. Solaris showcases pages with different UI designs and effects, placed on a single url. At the moment, it only showcases a 3D lightsaber... and a loading animation... Yeah.

Front End

  • HTML + jinja2 templates
  • CSS + Bootstrap 5
  • Javascript + JQuery and JQuery UI
  • Javascript libraries:
    • image cropping - Cropper.js
    • 3D rendering - Three.js for
    • emoji picker - Emoji-button.js
  • images from unsplash or created with photoshop
  • vector images from unDraw and other places

Back End

  • Language - Python
  • Framework - Flask
  • Some libraries:
    • Ckeditor
    • SQLAlchemy
    • WTForms
